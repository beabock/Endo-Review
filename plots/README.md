## Analysis Modes

### Main vs Supplementary
- **Main visualizations**: Exclude papers that focus only on mycorrhizal fungi (endophyte-focused analysis)
- **Supplementary visualizations**: Include all papers including mycorrhizal-only studies
- Both modes are available for all phylum-based taxonomic representations
- Each mode produces both count and percentage versions

### Visualization Types

1. **Taxonomic Representation** (`main/` and `supplementary/`)
   - Phylum-level breakdown of plant and fungal taxa
   - Hierarchical counting (includes species in genera, genera in families)
   - Both count and percentage representations
   - Consistent phylum ordering by species count

2. **Geographic Analysis** (`geographic/`)
   - Global research intensity and distribution patterns
   - Country and continent-level research coverage
   - Geographic-taxonomic diversity analysis
   - Temporal trends in research publication

3. **Data Extraction Quality** (`extraction/`)
   - Species detection rates and patterns
   - Research methods usage analysis
   - Plant parts studied frequency
   - Geographic information completeness
   - Prediction confidence analysis
   - Information quality heatmaps

4. **Model Evaluation** (root level)
   - Model performance metrics
   - Confusion matrices and learning curves
   - Feature importance analysis
   - Prediction confidence distributions

## File Naming Convention

### Main/Supplementary Plots
- `{kingdom}_{level}_representation_{mode}_by_phylum_{type}.png`
- Examples:
  - `plantae_species_representation_main_by_phylum_count.png`
  - `fungi_genus_representation_supplementary_by_phylum_percent.png`

### Geographic Plots
- Descriptive names indicating geographic scope and analysis type
- Examples: `geographic_taxonomic_diversity.png`, `kingdom_by_country.png`

### Extraction Plots
- Descriptive names indicating the analysis focus
- Examples: `species_detection_rate.png`, `methods_individual_core.png`

## Usage Notes

1. **For publications**: Use `main/` visualizations for primary figures, `supplementary/` for supplementary materials
2. **For geographic analysis**: Check `geographic/` folder for global research patterns
3. **For methodology assessment**: Review `extraction/` folder for data quality metrics
4. **For model evaluation**: Find model performance plots in the root level

## Scripts

- `scripts/04_analysis/visualization/visualize_taxa_results.R`: Generates main/supplementary and geographic visualizations
- `scripts/04_analysis/visualization/visualize_extraction_results.R`: Generates extraction quality visualizations
